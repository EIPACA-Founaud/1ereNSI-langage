L'HISTOIRE DU LANGAGE INFORMATIQUE

  • 1840 On peut définir plusieurs dates pour le début de la programmation. Mais retenons-en une : Ada Lovelace a défini en 1840 le principe des itérations successives dans l'exécution d'une opération. ‍
  • 1950-60 Le langage le plus utilisé est l’assembleur code de bas niveau, en 1955 arrive le langage Fortran chez IBM, en 1959, on a les prémices du langage COBOL.
  • 1970-80 Pascal, Prolog, SmallTalk, C, SQL, Awk, Ada, Rexx. Voici une liste des langages importants créés dans les années 70 la plupart des langues modernes comptent au-moins l'un d'entre eux dans son ascendance
  • Visual basic Lancé en 1991 un environnement de programmation dans lequel on pouvait créer une application en créant d'abord visuellement l'interface utilisateur, puis en ajoutant du code.
  • Python Python est un langage de programmation de haut niveau à usage général et largement utilisé crée en 1991
  • Ruby Le langage de programmation Ruby est un langage polyvalent hautement portable qui répond à de nombreux objectifs crée en 1993
  • PHP Il s'agit d'un langage de programmation à usage général utilisé pour concevoir un site Web ou une application Web.
  • Javalangage de programmation et une plate-forme informatique lancés pour la première fois par Sun Microsystems en 1995
  • Les années 2010 les noms associés à la programmation. Matlab et Scilab sont essentiels pour le calcul numérique R est le leader en statistiques. Ruby et Python ont connu une énorme croissance après 20 ans.
  • Conclusion En conclusion tous les langages informatiques ont révolutionnés le monde en commençant dans les années 1840 jusqu’à nos jours et ça n’est pas encore fini…
